# Totem::Lodestar
A gem built to include base functonaility for markdown based documentation rails app.

## Installation
Add this line to your application's Gemfile:
```ruby
gem 'totem-lodestar'
```

Then execute:
```bash
$ bundle install
```

```bash
$ rails g totem:lodestar:install
```
